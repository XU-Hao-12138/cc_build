# Love Journey - 恋爱路线记录平台

## 项目简介
一个记录和分享恋爱生活点滴的平台，页面酷炫浪漫。

## 技术栈
- 后端: Python FastAPI + SQLite + SQLAlchemy
- 前端: React 19 + TypeScript + Vite + Tailwind CSS 4 + Framer Motion

## 核心功能
1. 时间轴 - 恋爱历程展示
2. 记忆卡片 - 照片+文字记录
3. 里程碑 - 重要日子标记
4. 地图足迹 - 去过的地方
5. 分享页面 - 生成分享链接
